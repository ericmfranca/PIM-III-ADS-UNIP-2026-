# PIM-III-ADS-UNIP-2026-
Projeto acadêmico do PIM III – Análise e Desenvolvimento de Sistemas (UNIP), focado no desenvolvimento de uma plataforma web educacional para avaliação e apoio à aprendizagem, integrando engenharia de software ágil, banco de dados, C#, desenvolvimento web, UX/UI e análise de dados.
